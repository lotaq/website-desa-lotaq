## 🛠️ Teknologi

- **Framework**: [Astro](https://astro.build) — Static site generator modern
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Hosting**: [Cloudflare Pages](https://pages.cloudflare.com)
- **Domain**: lotaq.desa.id (PANDI)

## 📂 Struktur Project

```
website-kampung-lotaq/
├── public/              # Asset statis (favicon, gambar, dll)
├── src/
│   ├── components/      # Komponen Astro reusable
│   ├── data/            # Data resmi kampung
│   ├── layouts/         # Template halaman
│   ├── pages/           # Halaman website
│   └── styles/          # CSS global
├── astro.config.mjs     # Konfigurasi Astro
├── tailwind.config.mjs  # Konfigurasi Tailwind
└── package.json
```

## 🚀 Pengembangan Lokal

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build

# Preview hasil build
npm run preview
```
